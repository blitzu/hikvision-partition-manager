    - upgrade(): create all 9 tables in dependency order:
      1. locations
      2. nvr_devices (FK -> locations)

    For the GENERATED column in camera_disarm_refcount, use raw SQL in the migration:
    ```python
    op.execute("""
      ALTER TABLE camera_disarm_refcount
      ADD COLUMN disarm_count INTEGER
      GENERATED ALWAYS AS (cardinality(disarmed_by_partitions)) STORED
    """)
    ```
