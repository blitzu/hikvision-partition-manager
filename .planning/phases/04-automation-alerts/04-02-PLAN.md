  <behavior>
    stuck_disarmed_monitor:
    - Queries all Partition + PartitionState rows where state='disarmed' and alert_if_disarmed_minutes IS NOT NULL
    - For each: computes minutes_elapsed = (now - last_changed_at).total_seconds() / 60
    - If minutes_elapsed >= alert_if_disarmed_minutes: fires partition_stuck_disarmed webhook via asyncio.create_task(deliver_webhook(...))
    - One webhook per partition per cycle, indefinitely until rearmed
    - Session opened via async_session_factory(); closed after processing

    nvr_health_check:
    - Queries all NVRDevice rows (no filter on status), joins Location for location_name
    - For each NVR: calls ISAPIClient.get_device_info(); on success: sets status='online', last_seen_at=now; on exception: sets status='offline', last_seen_at unchanged
    - Tracks previous status in module-level dict `_nvr_prev_status: dict[uuid.UUID, str]`
    - Offline transition (prev != 'offline' and new == 'offline'): fires nvr_offline webhook
    - Online transition (prev == 'offline' and new == 'online'): fires nvr_online webhook
    - Cooldown: module-level `_nvr_last_offline_alert: dict[uuid.UUID, datetime]` — suppress nvr_offline webhook if last alert was within 5 minutes
    - DB commit after processing all NVRs (bulk update)

    Tests:
    - Test stuck_disarmed_monitor fires webhook for overdue partition, skips non-overdue
    - Test stuck_disarmed_monitor skips partitions with no alert_if_disarmed_minutes
    - Test nvr_health_check fires nvr_offline webhook on first failure
    - Test nvr_health_check suppresses nvr_offline webhook within 5-minute cooldown
    - Test nvr_health_check fires nvr_online webhook on recovery from offline
    - Tests use AsyncMock for DB session, ISAPIClient, and deliver_webhook
  </behavior>
  <action>
    1. Check `app/core/config.py` — if `ALERT_WEBHOOK_URL` is not already a field on `Settings`, add `ALERT_WEBHOOK_URL: str = ""` (empty default means webhooks are no-op when not configured). Also add `POLL_INTERVAL_SECONDS: int = 60` if missing.

    2. Create `app/jobs/monitors.py`:

       Module-level state:
       ```python
       _nvr_prev_status: dict[uuid.UUID, str] = {}
       _nvr_last_offline_alert: dict[uuid.UUID, datetime] = {}
       OFFLINE_ALERT_COOLDOWN_SECONDS = 300  # 5 minutes
       ```

       `async def stuck_disarmed_monitor() -> None`:
       - Open session: `async with async_session_factory() as db:`
       - Query: join Partition + PartitionState where PartitionState.state == 'disarmed' and Partition.alert_if_disarmed_minutes.is_not(None) and Partition.deleted_at.is_(None)
       - now = datetime.now(timezone.utc)
       - For each row: compute minutes_elapsed; if >= alert_if_disarmed_minutes: build payload and `asyncio.create_task(deliver_webhook(payload))`

       `async def nvr_health_check() -> None`:
       - Open session: `async with async_session_factory() as db:`
       - Query NVRDevice + Location (join via location_id); load all rows
       - For each NVR: create ISAPIClient using `decrypt_password(nvr.password_encrypted)`; attempt `await client.get_device_info()`
       - On success: new_status = 'online'; update nvr.status, nvr.last_seen_at = now
       - On exception: new_status = 'offline'; update nvr.status only
       - Check transition using `_nvr_prev_status.get(nvr.id, nvr.status)` as prev
       - If offline transition: check cooldown; if not in cooldown, fire nvr_offline webhook and update `_nvr_last_offline_alert[nvr.id] = now`
       - If online transition: fire nvr_online webhook
       - Update `_nvr_prev_status[nvr.id] = new_status`
       - `await db.commit()` after all NVRs processed

       Import `deliver_webhook` from `app.jobs.auto_rearm`. Import `async_session_factory` from `app.core.database`. Import `decrypt_password` from `app.core.crypto`. Import `ISAPIClient` from `app.isapi.client`.
